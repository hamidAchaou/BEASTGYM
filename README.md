## 🧠 Project Overview

**BEAST GYM** is a fully responsive fitness landing page built with semantic HTML, CSS, and vanilla JavaScript.  
It highlights modern UX/UI, SEO optimization, accessibility, and performance — all deployable with CI/CD via GitHub Actions.

---

## ⭐ Key Highlights

### 🎨 UI / UX
- Mobile-first and fully responsive layout
- Strong CTA design for conversions
- Smooth navigation with enhanced accessibility

### 📊 SEO & Structured Data
✔ Canonical URL  
✔ Meta description & keywords  
✔ Schema.org structured JSON-LD (HealthClub)  
✔ Optimized headings and semantic content

### ♿ Accessibility (A11y)
- Skip links for keyboard users
- ARIA roles & screen reader enhancements
- Reduced-motion support
- Logical tab order

### ⚡ Performance Improvements
- Preload critical fonts/styles
- Lazy-loaded images with dimensions
- Minimal JS and optimized scroll behavior
